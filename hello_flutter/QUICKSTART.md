# 快速开始指南

## 🚀 5分钟上手

### 第一步：检查环境

确保已安装 Flutter SDK：
```bash
flutter --version
```

如果未安装，请访问 [Flutter 官网](https://flutter.dev) 下载安装。

### 第二步：获取依赖

在项目根目录执行：
```bash
flutter pub get
```

### 第三步：连接设备

**Android 设备**：
1. 开启开发者模式和 USB 调试
2. 通过 USB 连接电脑
3. 运行 `flutter devices` 查看设备

**iOS 设备**：
1. 使用 Mac + Xcode
2. 信任电脑并连接设备
3. 运行 `flutter devices` 查看设备

**模拟器**（仅用于测试 UI，传感器不可用）：
```bash
# Android 模拟器
flutter emulators

# iOS 模拟器（Mac only）
open -a Simulator
```

### 第四步：运行应用

```bash
flutter run
```

应用将自动启动并在设备上运行。

## 📱 使用教程

### 首次使用

1. **授权权限**
   - 首次启动会请求相机权限
   - 点击"允许"以继续使用

2. **查看界面**
   ```
   ┌─────────────────────────┐
   │  [状态面板]              │
   │                         │
   │    [站位引导框]          │
   │                         │
   │                         │
   │      [拍照按钮]          │
   └─────────────────────────┘
   ```

3. **调整站位**
   - 站在距离手机 2-3 米处
   - 确保全身在引导框内
   - 双脚分开与肩同宽
   - 双手自然下垂

4. **观察状态**
   - 顶部状态面板显示4个指标
   - 红色 = 需要调整
   - 绿色 = 条件满足

5. **等待自动拍照**
   - 当所有指标变绿
   - 屏幕中央出现倒计时 3-2-1
   - 自动拍照

6. **确认结果**
   - 查看拍摄的照片
   - 点击"确认使用"或"重新拍摄"

### 手动拍照

如果自动拍照未触发，可以：
- 点击底部的相机图标按钮
- 随时手动触发拍照

### 常见问题

#### ❓ 相机黑屏
**原因**：权限未授予或相机被占用  
**解决**：
1. 检查是否授予相机权限
2. 关闭其他使用相机的应用
3. 重启应用

#### ❓ 状态一直是红色
**原因**：检测条件未满足  
**解决**：
1. 确保光线充足
2. 背景尽量简洁
3. 穿着与背景对比明显的衣服
4. 保持手机水平
5. 保持稳定不要晃动

#### ❓ 自动拍照不触发
**原因**：5个条件中有未满足的  
**解决**：
- 查看状态面板，找到红色的指标
- 根据提示调整
- 或使用手动拍照

#### ❓ 传感器数据不准确
**原因**：模拟器不支持传感器  
**解决**：
- 使用真机测试
- 确保设备有加速度计和陀螺仪

## 💡 提示

### 最佳拍摄实践

1. **环境**：
   - 光线充足但避免强光直射
   - 背景简洁，最好是纯色墙面
   - 地面平整

2. **站位**：
   - 距离手机 2-3 米
   - 正对相机
   - 双脚与肩同宽
   - 身体挺直

3. **持机**：
   - 双手握持保持稳定
   - 手机垂直地面
   - 镜头高度约1.2米
   - 轻微俯角（向下5-10度）

4. **服装**：
   - 穿着合身的衣物
   - 避免过于宽松
   - 颜色与背景形成对比

### 提高检测精度

- ✅ 确保全身在画面内
- ✅ 关键部位不被遮挡
- ✅ 保持静止2-3秒
- ✅ 手机保持水平
- ❌ 避免快速移动
- ❌ 避免复杂背景
- ❌ 避免逆光拍摄
